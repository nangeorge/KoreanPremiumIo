Required asset files for this project:
--------------------------------------

1. icon.png              - 1024x1024 app icon (no rounded corners — the OS applies them)
2. splash-icon.png       - 200x200 splash screen icon (centered on #0a0a0f background)
3. adaptive-icon.png     - 1024x1024 Android adaptive icon foreground
4. favicon.png           - 32x32 web favicon
5. notification-icon.png - 96x96 Android notification icon (white on transparent)
6. coin-placeholder.png  - 32x32 fallback coin logo (simple circle placeholder)

All icons should use the app's indigo (#6366f1) brand color.
The splash screen background color (#0a0a0f) is set in app.json.

You can generate these using:
  npx expo-splash-screen  (for splash screen)
  https://appicon.co      (for app icons)
  https://makeappicon.com (alternative)
